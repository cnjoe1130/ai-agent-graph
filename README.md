# AI Agent 生态概念图谱 2026

交互式力导向图，展示2026年AI Agent生态系统中各框架、协议、概念、基础设施和主要玩家之间的关系。

## 功能
- 🖱 拖拽节点固定位置
- 🔍 滚轮缩放
- 👆 悬停节点高亮关联 + 中英文详情
- 🔎 搜索框支持中英文

## 技术栈
- D3.js v7 力导向图
- 纯静态HTML/JSON，无需后端

## 数据
- 50+ 节点，覆盖框架、协议、概念、模型、玩家、基础设施、向量数据库、应用
- 每个节点含中英文描述和关系数据
